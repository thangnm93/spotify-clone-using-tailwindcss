## Demo

On Netlify [https://nextjs-spotify-clone.netlify.app](https://nextjs-spotify-clone.netlify.app)

## Install

```bash
npm install
# or
yarn install
```

## Run

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
